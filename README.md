## Getting Started

**Install**

```bash
npm install -g library-genesis-cli
```

**Execute CLI**

```bash
libgen ./relative/path/to/config.js
```

## Goals

* Implement [library-genesis](https://github.com/servexyz/library-genesis)
* Make it easier to spawn dynamic boilerplates
